== Description ==

This is a custom demo plugin where we create a custom post type 'Music' and taxonomy and list all posts on front using a short code `[music]`.

You can also pass the parameters to filter out the listing like 
`[music year=2021]`

A setting page also create to manage the currency and number of posts show on front on a single page.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `music.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[music]` in your templates
